# Proyecto OVR

## Descripción

Proyecto demostrativo desarrollado con Blazor y .NET para mostrar una arquitectura organizada y buenas prácticas de desarrollo de software.

## Tecnologías

- C#
- .NET
- Blazor
- ASP.NET Core
- SQL Server
- REST API

## Arquitectura

El proyecto está organizado utilizando separación de responsabilidades mediante:

- Domain
- Infrastructure
- Service API
- Client Services

## Objetivo

Este repositorio forma parte de mi portafolio profesional y tiene fines demostrativos.
